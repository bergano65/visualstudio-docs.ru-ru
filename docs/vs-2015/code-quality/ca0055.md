# <a name="ca0055"></a>CA0055

Не удалось загрузить файл:\<путь >.

Средство анализа кода не удалось загрузить файл, указанный для анализа. Это происходит, если файл не найден или был запрещен доступ к файлу. Как правило это происходит, если пользователь решил переопределить свойство входной сборки в Visual Studio, или вызвал fxcopcmd.exe вручную.
