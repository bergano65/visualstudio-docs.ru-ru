# <a name="c1254"></a>C1254
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Предупреждение C1254: несоответствие версий подключаемого модуля: версия не соответствует версии драйвера PREfast  
  
 Средство анализа кода сообщает это предупреждение при возникновении внутренней ошибки в версии подключаемого модуля, не в коде выполняется анализ.
