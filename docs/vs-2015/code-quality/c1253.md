# <a name="c1253"></a>C1253
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Предупреждение C1253: Не удалось загрузить файл модели.  
  
 Средство анализа кода сообщает это предупреждение при возникновении внутренней ошибки в файле модели, а не в коде выполняется анализ.
