# <a name="c1253"></a>C1253
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

предупреждение C1253: не удалось загрузить файл модели.  
  
 Средство анализа кода сообщает об этом предупреждении при возникновении внутренней ошибки в файле модели, а не в анализируемом коде.
