# <a name="c28351"></a>C28351
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Предупреждение C28351: Заметка \<Аннотация > описывает, когда динамическое значение (переменная) не может использоваться в условии.  
  
 Это предупреждение возникает в том случае, когда заметка применяется где проверяемой тип C/C++.  
  
 Выражение в `_When_` должны оцениваться как константы. `_When_` Учитывается.
