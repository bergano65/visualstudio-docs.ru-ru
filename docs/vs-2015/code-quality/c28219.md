# <a name="c28219"></a>C28219
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Предупреждение C28219: Для параметра аннотации ожидался член перечисления  
  
 Должен быть членом именованный параметр в заметку `enum` обнаружен тип и другой символ; используйте член, `enum` типа. Обычно это означает макроса неправильным кодом заметки.
