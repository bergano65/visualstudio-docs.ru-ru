# <a name="c28305"></a>C28305
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Предупреждение C28305: Ошибка при синтаксическом анализе \<token > была обнаружена.  
  
 Это предупреждение возникает в том случае, если выражение, содержащее указанный токен является некорректным.
