# <a name="exception-assistant-dialog-box"></a>Диалоговое окно "Помощник по исключениям"
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

**Исключениям** диалоговое окно отображается при возникновении исключения во время выполнения. **Исключениям** отображает тип исключения, Дополнительные сведения и ссылки на советы по устранению неполадок, предоставляет способ для получения дополнительной справки в Интернете и позволяет пользователю выполнять определенные действия, такие как просмотреть подробные сведения об исключении.  
  
## <a name="task-list"></a>список задач  
 Чтобы просмотреть раздел, посвященный устранению неполадок тип исключения, вы столкнулись, щелкните одно из сообщений подсказки, отображаемый в **советы по устранению неполадок** области.  
  
 Чтобы выполнить действия, связанные с исключением, щелкните одно из действий, показанных в области действий.  
  
 Сведения о том, как включить или отключить помощник по исключениям, см. в разделе [General, Debugging, диалоговое окно параметров](../debugger/general-debugging-options-dialog-box.md).  
  
## <a name="user-interface-element-list"></a>Список элементов пользовательского интерфейса  
 **Тип исключения**  
 Отображает тип вызванного исключения.  
  
 **Дополнительные сведения**  
 Отображает дополнительные сведения об исключении.  
  
 **Советы по устранению неполадок**  
 Отображает ссылки на советы по устранению неполадок, которые могут быть полезны для обнаружения источника исключения.  
  
 **Действия**  
 Перечисляет действия, которые можно выполнить, например просмотр дополнительной информации об объекте исключения.  
  
 **Справка в Интернете**  
 Позволяет выполнить поиск дополнительной справочной информации в Интернете.
