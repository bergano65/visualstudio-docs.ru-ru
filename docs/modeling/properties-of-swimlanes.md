# <a name="properties-of-swimlanes"></a>Свойства дорожек
Дорожки можно добавить в диаграмму. Дорожки разделить схему на вертикальной или горизонтальной области. Можно определить другие фигуры, для отображения внутри дорожки. Дополнительные сведения см. в разделе [способ определения доменного языка](../modeling/how-to-define-a-domain-specific-language.md). Дополнительные сведения об использовании этих свойств см. в разделе [Настройка и расширение доменного языка](../modeling/customizing-and-extending-a-domain-specific-language.md).

 Дорожки имеют свойства, которые перечислены в следующей таблице.

|Свойство.|Описание|Значение по умолчанию|
|-|-|-|
|Цвет заливки для тела|Цвет заливки для тела дорожки.|Белый|
|Цвет заливки заголовка|Цвет заливки для заголовка дорожки.|Темно-серый|
|Цвет разделителя|Цвет разделительной линии.|LightGray|
|Стиль линии разделителя|Стиль линии разделителя (`Solid`, `Dash`, `Dot`, `DashDot`, `DashDotDot`, или `Custom`).|`Dash`|
|Толщина разделителя|Толщина разделительной линии в дюймах.|0.03125|
|Цвет текста|Цвет, используемый для декораторов текста, которые связаны с этой дорожки.|Черный|
|Модификатор доступа|Уровень доступа класса (`public` или `internal`).|Public|
|Настраиваемые атрибуты|Используется для добавления атрибутов к классу код, созданный из этой дорожки.|\<none>|
|Создает двойной производным|Если `True`, будет создан базовый класс и разделяемый класс (для поддержки настройки посредством переопределений). Дополнительные сведения см. в разделе [переопределение и расширение созданных классов](../modeling/overriding-and-extending-the-generated-classes.md).|False|
|Имеет пользовательский конструктор|Если `True`, пользовательский конструктор будет предоставляться в исходном коде. Дополнительные сведения см. в разделе [переопределение и расширение созданных классов](../modeling/overriding-and-extending-the-generated-classes.md).|False|
|Модификатор наследования|Описывает тип наследования класса исходного кода, созданный из дорожки (`none`, `abstract` или `sealed`).|Нет|
|Базовая дорожка|Базовый класс для этой дорожки.|(нет)|
|name|Имя этой дорожки.|Текущее имя|
|Пространство имен|Пространство имен, которое входит в состав этой дорожки.|Текущее пространство имен|
|Тип подсказки|Определение всплывающей подсказки (`fixed`, `variable`, или `none`). Если `fixed`, то значение параметра `Fixed Tooltip Text` свойство используется; Если `variable`, а затем подсказка определяется в пользовательском коде.|\<none>|
|Примечания|Неофициальные заметки, связанные с этой дорожки.|\<none>|
|Выравнивание|Горизонтальное или вертикальное выравнивание.|Вертикально|
|Начальная высота|Начальная высота данной дорожки в дюймах. Применимо только к горизонтальным дорожкам.|0|
|Начальная ширина|Начальная ширина данной дорожки в дюймах. Применимо только к вертикальным дорожкам.|0|
|Предоставляет цвет текста|Если `True`, пользователь может задать цвет дорожку в созданном конструкторе. Для этого щелкните правой кнопкой мыши на фигуре дорожки и нажмите кнопку **добавить предоставленный**.|False|
|Описание|Используется для документирования созданного конструктора.|\<none>|
|Отображаемое имя|Имя, которое будет отображаться в созданном конструкторе для ссылки на этот класс swimlane.|\<none>|
|Фиксированный текст всплывающей подсказки|Текст, используемый для фиксированной подсказки.|\<none>|
|Ключевое слово Help|Ключевое слово, используемое для индексации справки F1 для этой дорожки.|\<none>|
