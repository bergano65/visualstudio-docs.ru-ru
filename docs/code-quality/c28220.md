# <a name="c28220"></a>C28220
Warning C28220: ожидалось целочисленное выражение для аннотации параметра в заметке

 Это предупреждение означает, что параметр в заметке должен быть целочисленным выражением, а обнаружен другой тип. Обычно это означает неправильное написание макроса аннотации.
