# <a name="select-code-type-dialog-box"></a>Диалоговое окно "Выбор типа кода"
Чтобы открыть это диалоговое окно, откройте диалоговое окно **Присоединение к процессу** и нажмите кнопку **Выбрать**.

 **Автоматически определять тип отлаживаемого кода** Соответствующий отладчик будет выбран в зависимости от вида выполняемого кода.

 **Отлаживать следующие типы кода:** В предоставленном списке выберите тип кода, который необходимо отладить.
