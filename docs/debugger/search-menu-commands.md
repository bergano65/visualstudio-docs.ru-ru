# <a name="search-menu-commands"></a>Команды меню "Поиск"
**Поиска** меню доступны команды для поиска конкретного окна.

 **Найти окно/процесс/поток/сообщение** открывает **окно**/**процесс**/**потоков** /  **Поиск сообщений** диалоговое окно, где можно использовать **средство поиска** или определение критериев поиска.

 **Найти далее** поиск в текущем представлении для следующего соответствующего окна, процесса, потока или сообщения. Данная команда меню (и связанные кнопки) доступна только при наличии допустимых результатов поиска, не является уникальным.

 **Найти ранее** поиск в текущем представлении предыдущего соответствующего окна, процесса, потока или сообщения. Данная команда меню (и связанные кнопки) доступна только при наличии допустимых результатов поиска, не является уникальным.
