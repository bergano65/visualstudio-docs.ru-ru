# <a name="search-tools-for-spy-views"></a>Средства поиска для представлений Spy++
Используйте эти диалоговые окна, чтобы указать, какие данные будут отображаться в окне представлений Spy++:

## <a name="in-this-section"></a>В этом разделе
 [Диалоговое окно "Поиск в окне"](../debugger/find-window-dialog-box.md) используется для просмотра свойств или сообщений из определенного окна.

 [Диалоговое окно "Поиск окна"](../debugger/window-search-dialog-box.md) используется для поиска узла конкретного потока в представлении окон.

 [Диалоговое окно "Поиск сообщений"](../debugger/message-search-dialog-box.md) используется для поиска узла для конкретного сообщения в представлении сообщений.

 [Диалоговое окно "Поиск процесса"](../debugger/process-search-dialog-box.md) используется для поиска узла конкретного процесса в представлении процессов.

 [Диалоговое окно "Поиск потока"](../debugger/thread-search-dialog-box.md) используется для поиска узла конкретного потока в представлении потоков.
