# <a name="edit-and-continue-dialog-box"></a>Изменить и продолжить - диалоговое окно
(Только для машинного кода) Данное диалоговое окно появляется, если в код были внесены изменения и операция **Изменить и продолжить** определила, что изменения кода могут быть применены. Дополнительные сведения см. в разделе [Изменить и продолжить](../debugger/edit-and-continue.md).

 **Никогда не показывать это диалоговое окно** Если этот флажок установлен, **изменить и продолжить** будет применять изменения кода без запроса разрешения в будущем. Данное предупреждение можно включить снова, если зайти в диалоговое окно **Параметры**, открыть папку **Отладка**, выбрать страницу **Изменить и продолжить**, а затем выбрать **Вызывать командами отладки** и **Запрос**.
