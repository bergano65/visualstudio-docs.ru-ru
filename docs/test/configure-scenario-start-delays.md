# <a name="configure-scenario-start-delays-in-load-tests"></a>Настройка задержки начала сценария в нагрузочных тестах

Задайте задержку перед запуском сценария в нагрузочном тесте с помощью редактора тестовой нагрузки и окна **Свойства**.

Свойство **Задержка времени начала** можно использовать, например, если требуется, чтобы один сценарий начал создавать элементы, используемые другим сценарием. Можно задержать сценарий-потребитель, чтобы сценарий-производитель успел заполнить некоторые данные.

Другой пример — сценарий, запускаемый только в определенное время дня. Для имитации такой ситуации требуется задержать запуск сценария.

## <a name="specify-the-delay-start-time-of-a-scenario"></a>Задание задержки времени начала сценария

Можно задать задержку запуска сценария в нагрузочном тесте, изменив с помощью редактора тестовой нагрузки значение свойства **Задержка времени начала** в окне **Свойства**.

> [!NOTE]
> Полный список свойств сценария тестовой нагрузки и их описание см. в разделе [Свойства сценария тестовой нагрузки](../test/load-test-scenario-properties.md).

Свойство **Задержка времени начала** можно использовать, например, если требуется, чтобы один сценарий начал создавать элементы, используемые другим сценарием. Можно задержать сценарий-потребитель, чтобы сценарий-производитель успел заполнить некоторые данные.

Другой пример — сценарий, запускаемый только в определенное время дня. Для имитации такой ситуации требуется задержать запуск сценария.

> [!NOTE]
> Полный список свойств параметров запуска и их описание см. в статье [Свойства сценария тестовой нагрузки](../test/load-test-scenario-properties.md).

### <a name="to-specify-the-delay-start-time-for-a-scenario"></a>Задание задержки времени начала сценария

1. Откройте нагрузочный тест.

     Открывается редактор тестовой нагрузки. Отображается дерево нагрузочного теста.

2. В папке **Сценарии** деревьев нагрузочных тестов выберите узел сценария, для которого требуется задать задержку времени начала.

3. В меню **Вид** выберите пункт **Окно свойств**.

     Категории и свойства сценария отображаются в окне **Свойства**.

4. В текстовом поле свойства **Задержка времени начала** введите значение времени, определяющее, через какое время после запуска нагрузочного теста будет запускаться данный сценарий во время выполнения нагрузочного теста.

    > [!NOTE]
    > Если для свойства **Отключить во время прогрева** этого сценария задано значение **True**, значение времени из свойства **Задержка времени начала** применяется после периода прогрева. Тем, какие сценарии входят в период прогрева, можно управлять с помощью свойства **Отключить во время прогрева** сценария.

5. После изменения этого свойства в меню **Файл** выберите команду **Сохранить**. После этого нагрузочный тест можно выполнять с новым значением параметра **Задержка времени начала**.

## <a name="enable-and-disable-whether-a-scenario-runs-during-the-warm-up-period"></a>Включение и отключение запуска сценария во время периода прогрева

Свойство **Отключить во время прогрева** задается в окне **Свойства**. Изменить свойства сценария тестовой нагрузки можно в редакторе тестовой нагрузки.

Свойство **Отключить во время прогрева** используется, чтобы указать, должен ли сценарий запускаться в период прогрева, задаваемый свойством **Задержка времени начала**. Дополнительные сведения см. в описании предыдущей процедуры [Задание задержки времени начала сценария](#specify-the-delay-start-time-of-a-scenario).

> [!NOTE]
> Полный список свойств для параметров запуска и их описание см. в статье [Свойства сценария тестовой нагрузки](../test/load-test-scenario-properties.md).

### <a name="to-enable-or-disable-the-warm-up-period-for-a-scenario"></a>Включение и отключение периода прогрева для сценария

1. Откройте нагрузочный тест.

     Отображается **редактор тестовой нагрузки**. Отображается дерево нагрузочного теста.

2. В папке **Сценарии** деревьев нагрузочных тестов выберите узел сценария, для которого требуется изменить поведение прогрева.

3. В меню **Вид** выберите пункт **Окно свойств**.

     В окне **Свойства** отображаются категории и свойства сценария.

     Для свойства **Отключить во время прогрева** выберите значение **True** или **False**.

4. По завершении изменения свойства в меню **Файл** выберите команду **Сохранить**. После этого нагрузочный тест можно выполнять с новым значением свойства **Отключить во время прогрева**.
