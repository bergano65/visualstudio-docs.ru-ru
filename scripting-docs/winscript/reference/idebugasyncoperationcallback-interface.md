# <a name="idebugasyncoperationcallback-interface"></a>Интерфейс IDebugAsyncOperationCallBack
Предоставляет состояния события, связанные с ходом оценки интерфейса `IDebugAsyncOperation`.  
  
## <a name="methods"></a>Методы  
 Помимо методов, наследуемых от `IUnknown`, `IDebugAsyncOperationCallBack` интерфейс предоставляет следующие методы.  
  
|Метод|Описание:|  
|------------|-----------------|  
|[IDebugAsyncOperationCallBack::onComplete](../../winscript/reference/idebugasyncoperationcallback-oncomplete.md)|Сообщает, что результат доступен из операции асинхронной отладки.|
