# IEnumDebugExtendedPropertyInfo::Reset
Сбрасывает последовательность перечислений в начало.  
  
## Синтаксис  
  
```  
HRESULT Reset (  
void  
);  
```  
  
## Возвращаемое значение  
 Возвращает допустимое `HRESULT`, обычно `S_ОК`.
